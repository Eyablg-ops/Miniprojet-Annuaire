# 🎓 Internship Platform - Student & Recruiter Management System

## 🚀 Project Overview

A comprehensive internship management platform connecting students, recruiters, and administrators. The system provides seamless authentication, profile management, and dashboard interfaces for all user types.

### User Types
- **👨‍🎓 Students**: Find and apply for internships
- **🏢 Recruiters**: Post opportunities and find talent
- **👑 Administrators**: Manage the platform and users

## 🛠 Technologies Used

### Backend (Spring Boot)
- Java 21
- Spring Boot 2.7.x
- Spring Security with JWT
- Spring Data JPA
- MySQL 8.0
- Maven

### Frontend (React)
- React 18
- React Router DOM
- Axios for API calls
- Modern CSS with animations

## ✨ Features

### 🔐 Authentication System
- **Student Registration**: Complete profile setup with education details
- **Recruiter Registration**: Company-linked registration
- **Admin Login**: Pre-configured admin access
- **JWT Token**: Secure authentication with token-based system
